This repository contains Data Structures and Algorithms practice exercises using JavaScript for beginners.

These DSA problems will help to improve your problem-solving skills and prepare for coding interviews.

## Problems Solved

### Arrays
- Get alternate elements
- Linear search
- Find largest element
- Find second largest element
- Find largest three elements
- Reverse an array
- Check if array is sorted
- Find missing number
- Find repeating number
  
### Pattern printing
- Pyramid
- Hollow square
- Hollow triangle
- Inverted triangle

## Goal

- Practice DSA daily
- Build strong problem-solving skills
- Improve coding interview preparation
